# Task 24: Cloud-Based Feature Implementation via Vibe Coding

## Feature: Cloud-Native Image Metadata Extractor
This feature was built using iterative Vibe Coding prompts to generate:
1. **REST API**: To upload and process image headers.
2. **Cloud Deployment**: AWS S3 and Lambda configuration scripts.
3. **Security**: IAM policy and environment variable management.

## Refinement Process
- **Prompt 1**: "Build a file uploader for S3."
- **Prompt 2**: "Add image validation and metadata extraction logic in Java."
- **Prompt 3**: "Secure the API with Spring Security and use AWS KMS for encryption."

## Conclusion
Vibe Coding reduces the manual effort of writing boilerplates and allows for rapid prototyping of complex cloud features while maintaining focus on high-level architecture.
