# Task 23: Vibe Coding & Prompt Engineering Implementation

## Experiment Log
- **Goal**: Generate a robust REST API using iterative prompt engineering.
- **Iteration 1**: "Create a spring boot api for student". (Result: Basic, no validation).
- **Iteration 2**: "Create a Spring Boot REST API for Students with JPA, MySQL, and @Valid annotations for email and name." (Result: Better, added validation).
- **Iteration 3 (Final)**: "Generate a production-ready Student REST API with Spring Boot 3, global exception handling, custom response DTOs, and JUnit tests." (Result: High quality, modular, and reliable).

## Evaluation
Iterative prompt refinement significantly improves:
1. **Accuracy**: Correct annotations and dependency usage.
2. **Scalability**: Proper architecture patterns.
3. **Security**: Validation and error handling included by default.
