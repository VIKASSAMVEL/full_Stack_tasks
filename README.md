## 🚀 Getting Started

### Prerequisites
- **Java**: JDK 17 or higher
- **Database**: MySQL Server
- **IDE**: IntelliJ IDEA, Eclipse, or VS Code
- **Optional**: Maven for build management

### Running the Tasks

#### Web/Frontend Tasks (e.g., Task 01, 03, 07)
Simply open the `index.html` or `login.html` file in any modern web browser.

#### SQL Tasks (e.g., Task 04, 05, 06)
1. Open your MySQL client.
2. Run the `schema.sql` or `.sql` files provided in the respective task folders to set up tables and test data.

#### Java/Spring Tasks (e.g., Task 10, 12, 13)
1. Import the project into your IDE.
2. Run the `Application` or `Main` class.
3. For REST APIs, use a tool like **Postman** or **cURL** to test the endpoints at `http://localhost:8080/api/...`.
